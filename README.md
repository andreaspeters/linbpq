# LinBPQ Lite

This is a hard fork of the original LinBPQ (git://vps1.g8bpq.net/linbpq) V23.

The reason for this fork is to reduce LinBPQ to Node and Convers functionality.

Right now I've deleted:

- BBS
- AIS
- BPQTerm
- PNG Encode/Decode
- MQTT
- FLDIGI
- SDRANGEL
- RTLSDR
- WinmorControl


## How to compile

```bash
make
```
